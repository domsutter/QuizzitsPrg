## Suggested exercises
1. Add a new `Deck` and a few cards; trace how `QuizManager` stores and loads them.
2. Change a question; run the app; confirm the UI updates.
3. Add a tiny feature (e.g., shuffle deck) and annotate your changes with comments.

> Tip: Try deleting a comment-block and rewriting it yourself as a comprehension check.
